BioChem\
Emergent Simulated Chemistry\
Discovered by @cactusjupiter\
This is a 4-state cellular automaton, using the Moore neighborhood, with chaining behavior resembling chemical bonds and reactions. Results in emergent stable mobile structures, sustained through chain reactions.\
